# NicoAIRanking_Claude_Skill

**Expert:** Nico AI Ranking
**Video Topic:** Build a Claude Skill for SEO Analysis
**Source:** youtube.com/@Nico_AIRanking

## Summary

Nico demonstrates how to build a custom Claude skill/agent that acts as a dedicated SEO analyst — connecting Google Search Console, GA4, and Google Business Profile data for integrated insights.

## Key Points

- **The SEO Analyst Agent**: A Claude-powered agent that ingests GSC search query data, GA4 traffic patterns, and GBP performance metrics to produce unified SEO recommendations.
- **Natural Language Querying**: Instead of navigating multiple dashboards, users ask questions like "which pages lost traffic this week?" and Claude queries the connected data sources.
- **Automated Reporting**: The agent can generate weekly SEO performance reports with trend analysis, anomaly detection, and prioritized recommendations.
- **Prompt Engineering for Accuracy**: Nico shares exact prompts for getting Claude to properly interpret SEO metrics — including error handling for API limitations and data gaps.

## Practical Takeaways

- Connect multiple analytics sources to Claude for unified SEO analysis
- Replace manual dashboard checking with natural language queries
- Automate weekly SEO reporting with AI agents
- Use specific prompts to ensure accurate data interpretation
